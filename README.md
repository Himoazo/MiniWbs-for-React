# Mini WMS 

Denna applikation är en REST API för ett minimalt lagerhanteringssystem.

API:et är utvecklad med .NET 9 och använder SQLite som databas. 

Detta API är publicerat [HÄR]() och konsumeras av en React applikation som är publicerad [HÄR]()

Detaljerad dokumentation om API:ets endpoints, requests och models finns [HÄR]() via Scalar.
